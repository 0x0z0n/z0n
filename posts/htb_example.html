Tags: Web, OSINT

Initial Enumeration

Performed full TCP scan using Nmap:

nmap -p- 10.10.10.10 -T4

Found open ports:

  • 22/tcp - SSH
  • 80/tcp - HTTP

Service Enumeration

Version detection scan:

nmap -sCV -p 22,80 10.10.10.10

Results showed OpenSSH 8.2 and Apache HTTP Server on port 80.
